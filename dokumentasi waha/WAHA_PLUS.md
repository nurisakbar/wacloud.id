# ➕ WAHA Plus

## 📋 Overview

**WAHA** didistribusikan dalam dua versi:

1. **Core** - versi dasar yang memenuhi hampir 80% kebutuhan pengguna. 100% gratis dan open source.
2. **Plus** - versi dengan fitur pesan lanjutan, keamanan, dan keandalan yang lebih baik.

👉👉 Bandingkan versi di halaman **🎁 Support Us**

## 💎 Tiers (Tingkatan)

Anda dapat mendukung proyek dan mendapatkan WAHA Plus dengan berdonasi ke proyek di salah satu platform yang tersedia.

👉👉 Lihat tier dan platform yang tersedia di halaman **🎁 Support Us**

## 🚪 Patron Portal

**Patron Portal**

Setelah berlangganan di Patreon atau Boosty, Anda akan mendapatkan akses ke **WAHA Patron Portal** -> di mana Anda akan mendapatkan password untuk mengunduh image Plus dan mengelola fitur-fitur Anda!

Baca lebih lanjut tentang Patron Portal di postingan **Patreon ->** atau **Boosty ->**.

### 📥 Install Plus

Setelah Anda mendapatkan password, dapatkan login Docker Hub Anda di Patron Portal -> dan jalankan perintah berikut:

```bash
docker login -u devlikeapro -p {KEY}
docker pull devlikeapro/waha-plus
docker logout
```

Kemudian di semua perintah, gunakan image **Plus** `devlikeapro/waha-plus` sebagai pengganti Core `devlikeapro/waha`.

### 👥 Invite Team (Undang Tim)

Jika Anda mendukung proyek dengan tier **WAHA PRO** - undang **hingga 5 anggota** ke tim Anda untuk membantu Anda mengembangkan proyek!

Anggota akan mewarisi tier **PRO** dan akan memiliki docker key mereka sendiri serta akses GitHub ke proyek.

**Portal - Tim Anda**

#### 📧 Undang Anggota

1. **Login** ke **Patron Portal**
2. **Buat Tim** di bagian bawah halaman
3. **Undang anggota baru** dengan mengirim undangan ke email mereka

#### ✅ Bergabung dengan Tim

1. Dapatkan **email dengan undangan**
2. **Login** ke **Patron Portal**
3. **Bergabung** dengan tim

> ⚠️ **Catatan:** Jika **Anggota** tidak melihat tier **PRO** - silakan refresh halaman setelah menunggu 10-30 detik (sinkronisasi tier membutuhkan sedikit waktu)
