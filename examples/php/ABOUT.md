WACloud adalah platform WhatsApp Gateway API yang powerful untuk developer. Dengan WACloud, Anda dapat dengan mudah mengintegrasikan WhatsApp ke dalam aplikasi Anda.

## 🌐 Link Resmi

- **Website**: [https://wacloud.id](https://wacloud.id)
- **Dashboard**: [https://app.wacloud.id](https://app.wacloud.id)
- **Dokumentasi**: [https://wacloud.id/docs.html](https://wacloud.id/docs.html)
- **YouTube**: [Channel WACloud](https://www.youtube.com/@wacloudid) - Tutorial dan panduan lengkap

## 📚 Fitur Utama

- ✅ RESTful API yang mudah digunakan
- ✅ Dukungan berbagai jenis pesan (text, image, video, document, poll, button, list)
- ✅ Webhook real-time untuk pesan masuk
- ✅ Multi-session management
- ✅ Template management
- ✅ OTP (One-Time Password) support
- ✅ Contacts management
- ✅ Rate limiting untuk keamanan
- ✅ Dokumentasi lengkap dengan contoh kode

## 🚀 Mulai Menggunakan

1. Daftar akun di [Dashboard](https://app.wacloud.id/register)
2. Buat API Key dari dashboard
3. Ikuti contoh kode di repository ini
4. Baca dokumentasi lengkap di [docs.html](https://wacloud.id/docs.html)
