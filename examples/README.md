## 🚀 Quick Start

1. **Pilih bahasa pemrograman** yang ingin Anda gunakan
2. **Baca README.md** di folder bahasa tersebut
3. **Install dependencies** sesuai dengan bahasa yang dipilih
4. **Copy config.example** menjadi config dan isi dengan API Key Anda
5. **Jalankan contoh kode** yang tersedia

## 📋 Fitur yang Tersedia

Semua contoh kode mencakup:

- **Device Management** - Membuat device, mendapatkan QR code, cek status
- **Send Text Message** - Mengirim pesan teks
- **Send Image Message** - Mengirim gambar dengan caption
- **Send Document Message** - Mengirim dokumen (PDF, DOC, dll)
- **Send Video Message** - Mengirim video dengan opsi video note
